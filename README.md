# Building Real-World GenAI Applications — Workshop Companion

Companion repository for the NPTEL+ workshop (July 2026, 3 sessions). Everything you need to revisit sessions, go deeper, and start building.

> **Sunset notice:** The companion *app* retires at the end of Week 3. This repository lives on, public and forkable. If you're reading this after July 2026, everything here still works — only the chat feature and cohort login are gone.

> **Note:** This is a personal educational project by the instructor, not an official IIT Madras or NPTEL+ product.

## Start here — pick your lane

| You are... | Start with |
|---|---|
| New to GenAI ("I've heard of ChatGPT") | [GLOSSARY.md](GLOSSARY.md), then [sessions/01-saathi/](sessions/01-saathi/) |

## What's in this repo

```
sessions/           Session recaps, slides, and links (one folder per session)
resources/          Curated podcasts, newsletters, websites, courses — tagged
templates/          Minimal, runnable starter templates (RAG, agent, full-stack)
decision-support/   Model decision matrix, tool comparison, cost heuristics
app/                Architecture of the companion app itself (a worked example)
```

Every resource entry carries tags: `[level] [effort] [domain] [free/paid]` — filter by what fits you.
