# Contributing

PRs are welcome **until the end of Week 3 (July 2026)**. After that this repo is archived-in-spirit: PRs may go unreviewed. Fork freely — the licenses guarantee you can.

## Good first contributions
- Fix a broken link or typo (`good-first-issue`)
- Add a resource to `resources/` — must include tags `[level][effort][domain][free/paid]` and a one-line "why I recommend this"
- Improve a template README's comments
- Add a row to the tool comparison (with a source link and date)

## Rules
1. **Content PRs**: keep the tag scheme; write plainly (many readers use English as a second language); no bare links — every link gets a "why".
2. **Template PRs**: declare upstream origin and license in the PR description AND the template's README. MIT-derived stays MIT; original work is GPLv3. The PR template has a checklist.
3. **Facts about models/costs**: include the source URL and a `verified: YYYY-MM-DD` date. Undated cost claims will be rejected — stale numbers mislead learners.
4. One PR = one concern.

## Process
Fork → branch → PR against `main`. Fill the PR template. Instructor reviews within the workshop window; review comments are teaching moments, treated as public discussion.
